# Public Sans font files

Licensed Public Sans family (Thin–Black, 9 weights, each with italic) uploaded 2026-07-08. `src/lib/fonts.ts` loads these `.woff2` files via `next/font/local`.

The `.woff`/`.ttf`/`.otf` copies aren't served by the site (everything in `public/` is web-accessible, and only woff2 is needed) — they live in `assets/fonts-source/` at the repo root for design-tool/reference use instead.
